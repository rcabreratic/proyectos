Descripción del vídeo recursos alternativos

Este vídeo comienza haciendo una pequeña introducción sobre la importancia de los recursos en Android (cadenas de texto, imágenes, vistas...) y cómo se identifican mediante un identificador único o ID de forma que se podrá acceder a ellos. Cuando no se especifica ninguna condición para esos recursos, hablamos de recursos por defecto. Cuando queremos que dada una configuración especial (orientación, idioma, versión API etc) se muestre un recurso u otro hablamos de recurso alternativo. Este recurso se debe almacenar en una carpeta con un sufijo que indique cuándo ha de ser utilizado. A continuación explica los ejemplos más comunes: orientación del terminal, idiomas, densidad de pantalla etc.
