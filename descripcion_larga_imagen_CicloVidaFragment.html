Ciclo de vida de un Fragment

En esta imagen se muestra un diagrama de flujo con el texto en inglés, que describe el ciclo de vida de un fragment.

  1. Fragment is added.

    1. onAttach()
    2. onCreate()
    3. onCreateView()
    4. onAtivityCreate()
    5. onStart()
    6. onResume()

  2. Fragment is active.

    • The fragment is added to the back stack, then removed/replaced.
      1. onPause()
      2. onStop()
      3. onDestroyView()
        1. The fragment returns to the layout from the back stack (go back to onCreateView())
    • User navigate backward or fragment is removed/replaced
      1. onPause()
      2. onStop()
      3. onDestroyView()
      4. onDestroy()
      5. onDetach()

  3. Fragment is destroyed.
