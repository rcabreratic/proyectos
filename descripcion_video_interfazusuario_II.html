Descripción del vídeo Crear una Interfaz II

Inicio

Este vídeo es la continuación del vídeo anterior, donde se explicó cómo crear una interfaz en código Java. En este vídeo se utilizará Android Studio y el lenguaje XML y partimos del ejemplo que se realizó. Partimos del método setContentView() de la clase Activity que indica la vista que tiene que mostrar y descomentamos la siguiente línea de código:

setContentView(R.layout.activity_main);

Comprobamos que compila ejecutando la aplicación mediante la opción Run -> Run app.

Para localizar la clase R, se tiene que cambiar el panel que hay a la izquierda de Android Studio donde inicialmente pone Android por la opción Project. Vamos al directorio app - > build -> generated -> source -> r -> debug y vemos el archivo R. Cada clase es un recurso que carga en código Java. 

Vamos a la opción Edit -> Find -> Find y buscamos el recurso activity_main y veremos el identificador de este recurso que se crea automáticamente, si se modifica, cuando se compile el recurso tendrá otro identificador con lo cual este fichero R no debe modificarse nunca manualmente. 

Volvemos a la vista Android y vemos el contenido del fichero activity_main.xml que incluye etiquetas XML. Dependiendo de la versión de Android Studio el contenido de este fichero puede variar. Este fichero se puede modificar tanto de la pestaña Text o desde la pestaña Design. En la vista Text, el autor explica la ventaja de usar la etiqueta <include> que permite reutilizar el código de la interfaz. 

Para incluir un comentario se puede realizar pulsando la tecla de acceso rápido Ctrl+z. 

En la vista Design o diseño es bastante fácil trabajar arrastrando los componentes desde el panel Components Tree que muestra todos los controles gráficos separados por categorías. Cuando se selecciona en la vista un objeto se muestra a la derecha el panel propiedades o Properties que es muy intuitivo ya que evita aprender todos los atributos de un componente. Finalmente el autor comprueba cómo lo que se ha modificado en modo diseño aparece en el panel del texto. 

Para eliminar un componente es fácil ya que se selecciona el elemento y se pulsa sobre la tecla Supr, y propone agregar un componente del tipo Button. Selecciona este componente dentro del Editor y lo arrastra modificando por tanto la propiedad align y la propiedad margin del componente.
