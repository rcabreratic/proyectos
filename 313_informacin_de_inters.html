3.1.3.- Información de interés

Recomendación

Recursos necesarios:

  • Ordenador personal (min. 16GB RAM).
  • SDK de Android (API 34)
  • Android Studio (Iguana | 2023.2.1 o posterior)

Recomendaciones:

  • Al crear el proyecto, utilizad como API mínima  la 24.
  • Tened en cuenta siempre las indicaciones que se dan para cada apartado y, en caso de dudas, consultad al profesor.
