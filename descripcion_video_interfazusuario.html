Crear interfaz de usuario I y II

En ambos vídeos se explica cómo crear una interfaz de usuario en Android Studio. En el primer vídeo el autor se centra en explicar las opciones del editor Android Studio en base a un proyecto en blanco con una Activity y su Lavout. A la hora de crear una inerfaz se tienen dos vistas: modo texto mediante la pestaña Text o gráficamente mediante la pestaña Design. Explica cómo escribir comentarios dentro de una clase Java y cuáles son los métodos que hay que definir dentro de una Activity aprovechando el Autocompletadodel editor que nos evita teclear por ejemplo el nombre de un método entero.

Una vez que en el primer vídeo se conoce un poco el entorno de desarrollo, explica cuáles son los controles básicos de una interfaz que se encuentran dentro del panel Palette del editor. De forma que se arrastra el componente al editor y se modifican sus atributos utilizando el panel Attributes.

Para conocer los métodos de un objeto el autor del vídeo hace referencia varias veces a la documentación oficial de Android Studio así como para entender la herencia de los controles básicos o View.

En el segundo vídeo el autor se centra en explicar los recursos de un proyecto y en concreto cómo a través de la clase R.class que podemos acceder a los recursos que se han definido en un Layout. Cuando se quiera reutilizar una interfaz ya definida el segundo vídeo nos explica cómo usar include de forma que se hace referencia a un Layout ya definido.Finalmente se puede comprobar su ejecucuíón en un emulador.
